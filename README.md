# install
prepare Rust lang environment
```shell
git clone this_repository
mv MHV4_monitor
cargo build (-r)
```

# test
```shell
cargo test -- --nocapture
```

# usage
generate web server at localhost:8080

```shell
cargo run (-r) 
```
