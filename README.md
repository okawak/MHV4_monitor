# install
prepare Rust lang environment. see https://www.rust-lang.org/tools/install
```shell
git clone https://github.com/okawak/MHV4_monitor.git
cd MHV4_monitor
```

# test
```shell
cargo test -- --nocapture
```

# usage
generate web server at 0.0.0.0:8080

```shell
cargo run
# or
./run.sh
```
